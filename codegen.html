MAQE Homework Challenge - Code Generation

Time: about 60 minutes recommended.

In this test your code and implementation will be evaluated. Please try to program as you normally would when maintaining a piece of software.

You may choose any programming language to do this test in however please double check with us beforehand if you decide to use a language other than PHP, Python, Node.js or Ruby.

Please do not use any large frameworks, such as Laravel or Codeigniter, for this test.

Please create a program that when run creates a unique list of coupon codes. The program has the following acceptance criteria.

It must:

  • create 250 codes
  • the codes must be 10 characters long
  • the codes must begin with "MQ"
  • write the 250 codes to a file

When you have finished this task please generate the 250 codes into a file called "section2".

Feel free to use the internet to look up any coding references, but please do not ask anyone else for help. We want to see how you code.

When you have finished all required tests email your completed files (including your resume and cover letter) to recruit@maqe.com.

Tests by MAQE.
