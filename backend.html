Server-side Test

In this test your code and implementation will be evaluated. Please try to program as you normally would when maintaining a piece of software. During this test you may use the internet, stackoverflow, and any other resource you normally use during development with the exception of asking friends for help during this test.

You may choose any programming language to do this test in however please double check with the interviewer beforehand if you decide to use a language other than php, python, node js or ruby.

Please do not use large frameworks such as laravel or codeigniter for this test.

Please create a program that when run creates a unique list of coupon codes. The program has the following acceptance criteria.

It must:

  • create 250 codes
  • the codes must be 10 characters long
  • the codes must begin with "MQ"
  • write the 250 codes to a file

When you have finished this task please generate the 250 codes into a file called "section2".

When you have finished email your completed file(s) to foobar@maqe.com.
