Javascript Test

Time: about 30 minutes.

Succesfully render a list of people in a web page using JSON data and a Javascript templating library of your choice (like Handlebars.js).

Here is the JSON data file:

https://raw.github.com/maqe/maqe/gh-pages/persons.json

And use the following HTML structure to create your template:

  <li class="person">
    <img src="http://images/goes/here.jpg" />
    <div class="name">First Name (Nickname)</div>
    <div class="title">Title</div>
  </li>

Note: Only show a "nickname" if the person has one.

Do as much as you can in the time you have. Don't worry if your work is not 100% exact or complete.

When you have finished email your completed file(s) to foobar@maqe.com.

You may use the internet to look up any Javascript, HTML or CSS references, but do not ask anyone else for help.
