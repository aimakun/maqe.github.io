MAQE Homework Challenge - Template and Styling

Time: about 60 minutes recommended.

Recreate the following reference image in HTML and CSS by using a templating library.

You can use any markup, styles and layout method necessary, but you must use these JSON data files: authors and posts.

If you are applying as a Server-Side Engineer definitely use a server-side language. (MAQE users PHP primarily, but feel free to use whichever language you're most comfortable with.) Frameworks are OK too.

If you are applying as a Client-Side Engineer please use Javascript and a JS templating library (like Handlebars.js, etc.) Frameworks are OK too.

If you are applying as a a Fullstack Engineer feel free to do one or both versions.

Click the image for full-size version.

Feel free to use the internet to look up any coding references, but please do not ask anyone else for help. We want to see how you code.

When you have finished all required tests email your completed files (including your resume and cover letter) to recruit@maqe.com.
